GM Media Tags is a plugin for easy managment of media taxonomies

== Description ==

Register a taxonomy 'Media Tag' for attachment and gives the ability to bulk assignment attachment taxonomies in backend, even for taxonomies not registered by plugin.

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.3 or greater
* MySQL version 5.0 or greater
